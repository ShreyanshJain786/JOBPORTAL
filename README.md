# JOBPORTAL
Welcome to the Job Portal project! This repository contains a fully functional job portal application designed to connect job seekers with potential employers. The platform offers a comprehensive solution for job listings, applications, and candidate management, providing an intuitive and seamless experience for both job seekers and recruiters.

Features
Job Seeker Functionality:

User registration and profile management
Job search and filter options
Job application submission
Resume upload and management
Personalized job recommendations
Employer Functionality:

Employer registration and profile management
Job posting and management
Search and filter candidates
Communication with applicants
Admin Panel:

User and role management
Job and application moderation
Analytics and reporting
General Features:

Responsive and mobile-friendly design
Secure authentication and authorization
Real-time notifications
Multi-language support

**Technology Used** 
**Frontend:**

>React
>Tailwind CSS

**Backend:**

>Node.js
>Express
>MongoDB
>

>JWT
